O Segredo do Controle Térmico

Sobre Nós.


Nós somos a SensSky, uma empresa dedicada aproporcionar uma mudança no setor da aviação.
Entregando uma solução que oferece redução de custos e maior segurança para voos comerciais, monitorando o superaquecimento de turbinas nas frotas de aeronaves.

Serviços.




Monitoramento dos Sensores

A temperatura dos motores dos aviões será capturada atraves de um sensor ligado a um Arduino que ira armazenar esses dados.


Armazenamento dos dados

Os dados seram armazenados em um banco de dados para que posteriormente o cliente consiga fazer a vizualização dos mesmos.


Análise comportamental

Com os dados capturados e armazenados será possível uma análise do comportamento da temperatura dos motores dos aviões atraves do dashboard.
